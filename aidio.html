GO LIVE

Step-by-step guide to publish this exact website for free

What you need

A free GitHub account, a free Netlify or Vercel account, and a free Anthropic API key. No coding experience needed. Total time: about 20 minutes.

1

Get your free Anthropic API key

Go to console.anthropic.com → Sign up free → Go to "API Keys" → Create a new key. Copy it. This is what lets your website call the AI. New accounts get free credits to start.

2

Download this website file

This entire website is one single HTML file. Download it and open it in any text editor (like Notepad or VS Code). Find the line that says YOUR_API_KEY_HERE and replace it with your real API key.

3

Create a free GitHub account

Go to github.com → Sign up free → Create a new repository → Upload your HTML file. GitHub stores your code for free.

4

Deploy on Netlify (free)

Go to netlify.com → Sign up free → Click "Import from Git" → Connect your GitHub → Select your repo → Click Deploy. In 60 seconds your site is live at a free URL like yoursite.netlify.app

5

Optional: get a custom domain

Buy a domain at namecheap.com (about $10/year). In Netlify settings, go to "Domain management" and add your domain. Your site will be at yourdomain.com instead.

⚠️

Important: protect your API key

For a public website, never put your API key directly in the HTML file — anyone can see it. Instead, use Netlify's free serverless functions to keep the key secret on the server. This is a slightly more advanced step. Netlify docs explain it clearly →
